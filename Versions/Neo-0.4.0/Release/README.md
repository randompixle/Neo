Neo lets you use apt-like commands on distros that aren't using apt.
Works great on Bazzite/Fedora (rpm-ostree) with Flatpak fallback.

## Install
```bash
./install.sh
neo sys-check
```

## Update
```bash
neo self-update
```

## Uninstall
```bash
neo uninstall-self
# or
./uninstall.sh
```
